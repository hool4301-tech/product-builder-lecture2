
# **Lotto Number Generator**

## **Overview**
A simple, user-friendly web application to generate random lottery numbers. The application will be a single-page interface built with HTML, CSS, and JavaScript, following modern web development best practices.

## **Design and Features**
### **Visual Design**
*   **Layout:** A clean, centered layout that is mobile-responsive.
*   **Typography:** Using 'Poppins' from Google Fonts for a modern and readable look.
*   **Color Scheme:** A vibrant gradient background with a contrasting color for the text.
*   **Iconography:** A lottery-themed emoji icon (🎰) in the header.
*   **Interactivity:** An improved "Generate" button with a hover effect, and an animation for the generated numbers.
*   **Effects:** A soft `box-shadow` on the main container to give it a "lifted" look. A subtle noise texture on the background.

### **Functionality**
*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** The generated numbers are displayed one by one with an animation to create a sense of anticipation.
